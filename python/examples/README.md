# Python Examples

This directory contains experimental or instructional scripts built on top of
the `isabelle_repl` Python client.

These files are not part of the installable package API. They are intended to:

- demonstrate how to compose the low-level REPL primitives
- provide a staging area for proof-repair experiments
- avoid coupling evolving agent logic to the stable client package

Current examples:

- `proof_repair_demo.py`: a prototype repair loop with a pluggable repair hook
